# DE1-SoC GPIO pinout

This document consists of a detailed overview of the GPIO pins on the DE1-SoC board

![GPIO_headers](https://github.com/user-attachments/assets/055184c5-1858-4b62-a64c-cbff7e354254)

*Special pins:*
- Pin 11: VCC5   - DC +5V
- Pin 12: GND
- Pin 29: VCC3P3 - DC +3.3V
- Pin 30: GND
