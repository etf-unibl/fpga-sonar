◆ PROCESS_4()

PROCESS_4 (   angle_i  
)
Process

Process for calculating duty cycle.

Computes the duty cycle based on the input angle using linear interpolation. The formula used is: duty_cycle = c_MIN_COUNT + angle * c_CYCLES_PER_STEP

◆ PROCESS_5()

PROCESS_5 (   clk_i ,
  rst_i  
)
Process

Process for generating the PWM output signal.

  • Counts clock cycles for each PWM period.
  • Sets the PWM output high ('1') when the counter is less than the duty_cycle, otherwise sets it low ('0').
  • Supports asynchronous reset via rst_i.
